## 删除 $WINDOWS.~BT
step1:
```shell
takeown /F C:\$Windows.~BT\* /R /A
```
step2:
```shell
icacls C:\$Windows.~BT\*.* /T /grant administrators:F
```
step3:
```shell
rmdir /S /Q C:\$Windows.~BT\
```

## 创建redis服务
安装服务
```shell
redis-server --service-install redis.conf
```
启动服务
```shell
redis-server --service-start
```
检查服务状态
```shell
redis-server --service-status
```
